📘 Developer Notes – ASCVD Risk & Scores

📌 Purpose

This page explains how to modify or update calculation logic for interactive medical scores such as the ASCVD Risk Calculator.

🧮 How ASCVD Calculator Works

The ASCVD calculator uses an external JSON file to load coefficients (weights) for various risk factors.

File location: /scores/ascvd_coefficients.json

🛠 How to Edit the Coefficients

  1. Open the file ascvd_coefficients.json using any text editor.
  2. Edit the values for different risk groups (male, female, smoker, etc.).
  3. Save and refresh your browser. No need to modify HTML or JavaScript.

📂 File Structure Summary

  • /scores/ascvd_risk_with_json.html – the calculator interface
  • /scores/ascvd_coefficients.json – all the math data
  • /assets/js/theme-toggle.js – handles light/dark mode
